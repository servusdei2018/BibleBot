# Security Policy

## Supported Versions

These versions of BibleBot are currently supported:

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| 1.0.x   | :x:                |

## Reporting a Vulnerability

To report a vulnerability, please open a new issue [here](https://github.com/servusdei2018/BibleBot/issues).
